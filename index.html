Tecnologías Básicas de la Web

  • HTML5
  • CSS3
  • JavaScript

Resumen Informativo

Las tres tecnologías trabajan en conjunto para crear la experiencia que conocemos como "página web". Se pueden resumir así:

HTML5 define el contenido y la estructura (los cimientos y muros). CSS3 define la presentación y el diseño (la pintura y decoración). JavaScript define el comportamiento y la lógica (la electricidad y tuberías).

Cuestionario de Temática

  1. ¿Cuál es la función principal de HTML5?
    • a) Programar la base de datos.
    • b) Definir la estructura y el contenido.
    • c) Dar estilo y colores.
  2. ¿Qué tecnología se encarga del diseño responsivo?
    • a) CSS3.
    • b) HTML5.
    • c) JavaScript.
  3. ¿Qué representa JavaScript en la analogía de la casa?
    • a) Los cimientos.
    • b) La pintura.
    • c) Las instalaciones funcionales.
  4. ¿Qué significa que HTML sea un lenguaje de marcado?
    • a) Que usa etiquetas para estructurar datos.
    • b) Que es un lenguaje para videojuegos.
    • c) Que solo marca errores.
  5. ¿Cuál es un lenguaje de programación real?
    • a) HTML5.
    • b) JavaScript.
    • c) CSS3.

Cuestionario de Etiquetas

  1. ¿Para qué sirve la etiqueta <nav>?
    • a) Para videos.
    • b) Para agrupar los enlaces del menú.
    • c) Para buscar en la web.
  2. ¿Qué atributo es obligatorio para la accesibilidad en imágenes?
    • a) src.
    • b) alt.
    • c) width.
  3. ¿Qué etiqueta se usa para incrustar un video de YouTube?
    • a) <video>.
    • b) <iframe>.
    • c) <embed>.
  4. ¿Cuál es la etiqueta para el título principal?
    • a) <h6>.
    • b) <header>.
    • c) <h1>.
  5. ¿Qué etiquetas crean una lista con viñetas (puntos)?
    • a) <ul> y <li>.
    • b) <ol> y <li>.
    • c) <list> y <item>.
