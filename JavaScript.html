¿Qué es JavaScript?

JavaScript es el lenguaje de programación encargado de la interactividad. Es lo que permite que una página web haga algo más que solo mostrar información estática: desde galerías de imágenes que se deslizan, hasta actualizaciones de contenido en tiempo real sin recargar la pestaña.

Hitos de JavaScript (Para validar)

  • 1995: Brendan Eich crea el lenguaje en solo 10 días para el navegador Netscape.
  • 2009: Lanzamiento de Node.js, permitiendo que JavaScript se use también en servidores (fuera del navegador).
  • 2015: Lanzamiento de ES6 (ECMAScript 2015), la mayor actualización que modernizó toda su sintaxis.

Comparativa: JS Antiguo vs JS Moderno (ES6+)

Característica JavaScript Tradicional JavaScript Moderno (ES6+)
Variables Uso de var (propenso a errores). Uso de let y const (más seguro).
Funciones Funciones clásicas function(). Funciones de flecha () => (más cortas).
Asincronismo Uso complejo de Callbacks. Uso de Promises y Async/Await.
