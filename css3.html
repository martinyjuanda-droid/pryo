¿Qué es CSS3?

CSS3 es la evolución de las Hojas de Estilo en Cascada. A diferencia de las versiones anteriores, CSS3 está dividido en "módulos", lo que permite que el lenguaje evolucione más rápido. Es el encargado de separar el contenido (HTML) de la presentación visual, permitiendo crear sitios atractivos y adaptables a cualquier pantalla.

Hitos de CSS3 (Para validar)

  • 1999: Comienza la creación de los borradores de CSS3 divididos por módulos.
  • 2011: Se estandarizan los módulos de Selectores, Espaciado y Colores (inicio del diseño moderno).
  • 2017: La llegada de CSS Grid revoluciona la forma de crear maquetaciones complejas.

Comparativa: CSS 2.1 vs CSS3

Característica CSS 2.1 CSS3 (Actual)
Diseño Basado en tablas o floats (difícil). Flexbox y Grid (mucho más sencillo).
Decoración Bordes cuadrados y colores planos. Bordes redondeados, sombras y degradados.
Dispositivos Diseño fijo para computadoras. Media Queries (Diseño Responsivo).
